# React Quiz App

## Run Locally

Clone the project

```bash
  git clone git@github.com:EidmantasDev/react-place-picker.git
```

Go to the project directory

```bash
  cd react-place-picker
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
